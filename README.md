# README

We are building an MCP server in Swift for Apple Reminders.

We are implementing the following tools:

## Lists

- `list_lists`
- `create_list`
- `update_list`
- `delete_list`
- `get_list`

## Reminders

- `list_reminders`
- `get_reminder`
- `create_reminder`
- `update_reminder`
- `delete_reminder`

We are supporting all the fields for each tool, such as: notes, priority, due date, repeat schedule, etc.

It's also possible to attach URL, image, or file attachments to reminders.
